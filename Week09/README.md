#  如何啟動後端如何啟動後端（npm install、npm run dev）。
- 下載node.js和nvm For Windows，在cmd中載好nvm版本
- 到VS code中mkdir Week09 && cd Week09、mkdir server && cd server
- 新增相關檔案，在VS code 的cmd中輸入npm install、npm run dev
